# Alcance del Proyecto: BeautyBook

## Descripción General
BeautyBook es una aplicación web diseñada para digitalizar y simplificar el agendamiento de citas en barberías y salones de belleza, optimizando el tiempo tanto de los clientes como de los profesionales.

## Usuarios Objetivo (Roles)
1. **Cliente:** Busca servicios, selecciona profesionales y agenda citas de forma autónoma.
2. **Administrador/Profesional:** Revisa su agenda diaria, gestiona las citas (confirmar/cancelar) y actualiza su catálogo de servicios.

## Funcionalidades Clave (MVP)
* **Catálogo de Servicios:** Visualización clara de cortes, arreglos, etc., con precio y duración estimada.
* **Flujo de Agendamiento:** Proceso por pasos: 
  1. Elegir servicio.
  2. Elegir profesional.
  3. Elegir fecha y hora disponible.
  4. Confirmación.
* **Gestión de Agenda:** Panel visual para que el profesional vea sus citas del día.
