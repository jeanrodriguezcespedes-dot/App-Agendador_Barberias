# Sistema de Diseño: BeautyBook

## Enfoque de Diseño
* **Mobile-First:** La interfaz debe estar diseñada primero para pantallas móviles (celulares), escalando elegantemente a tablets y escritorio.
* **Estilo:** Moderno, elegante, minimalista (Clean UI).

## Tipografía
* **Fuente Principal (Títulos y botones):** `Montserrat` (Sans-serif) - Transmite elegancia y modernidad.
* **Fuente Secundaria (Cuerpos de texto):** `Roboto` o `Open Sans` (Sans-serif) - Excelente legibilidad.

## Paleta de Colores
* **Fondo Principal:** `#121212` (Gris Carbón / Casi Negro) - Da un aspecto premium.
* **Fondo Secundario (Tarjetas/Formularios):** `#1E1E1E` (Gris Oscuro).
* **Acento Principal (Botones, íconos activos):** `#D4AF37` (Dorado Clásico) - Contraste elegante con el fondo oscuro.
* **Texto Principal:** `#F5F5F5` (Blanco Humo).
* **Texto Secundario:** `#A0A0A0` (Gris Claro).

## Reglas de Espaciado (Espaciado Base)
* Sistema basado en múltiplos de 8px (8px, 16px, 24px, 32px) para paddings y margins, asegurando consistencia visual.
