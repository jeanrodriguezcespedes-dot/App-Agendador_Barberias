# Stack Tecnológico

Este proyecto utiliza una metodología de desarrollo incremental.

## Fase 1: Maquetación y Estructura (Actual)
* **HTML5:** HTML semántico (`<header>`, `<nav>`, `<main>`, `<section>`).
* **CSS3:** * Uso de Flexbox y CSS Grid para layouts.
  * Uso de Variables CSS (Custom Properties) para los colores del Design System.
  * *No* se utilizarán frameworks como Bootstrap o Tailwind en esta fase.
* **Control de Versiones:** Git y GitHub.

## Fase 2: Lógica (Próximamente)
* **JavaScript (Vanilla):** Manipulación del DOM, manejo de eventos y validación de formularios base.

## Fase 3: Framework y Backend (Futuro)
* **React 18+:** Migración de la interfaz a componentes funcionales.
* **Manejo de Estado:** React Hooks (`useState`, `useEffect`).
* **Backend as a Service (BaaS):** Firebase (Firestore para base de datos de citas y Authentication para login de usuarios).
