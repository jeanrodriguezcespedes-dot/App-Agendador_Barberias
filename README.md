# 💈 BeautyBook - Sistema de Agendamiento

Aplicación web para agendar citas en barberías y salones de belleza. 

## 🚀 Fases del Proyecto
- [ ] Fase 1: Maquetación y estructura semántica (HTML/CSS)
- [ ] Fase 2: Lógica e interactividad (JavaScript)
- [ ] Fase 3: Componentización y Base de Datos (React + Firebase)

## 🛠️ Cómo ejecutar
Abre el archivo `index.html` en tu navegador o usa Live Server.
